# Design System: Dribbble Creative Dev

## Overview
A cutting-edge, highly visual design system for a creative technologist. This system uses a deep `#050505` void as the canvas and punches through it with massive `Space Grotesk` typography and an unmistakable Acid Green (`#D4FF00`) accent. 

## Rules
- **No identical cards.** Use asymmetrical, masonry, or uniquely proportioned sections.
- **Huge Type.** Make headlines obnoxiously large to serve as graphical elements themselves.
- **Acid Accents.** The primary green is the only color allowed. Everything else is grayscale.
- **Single Page Flow.** Sections blend into one another using scroll-driven spacing and massive typographic section headers.
